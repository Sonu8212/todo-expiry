# Changelog

## [1.0.0] - 2026-06-29

### Added
- Green / yellow / red highlighting based on TODO due date
- Hover tooltip showing days remaining or overdue count
- Status bar item showing overdue / due soon / on track counts
- Manual refresh command: `TODO Expiry: Refresh Decorations`
- Configurable warning days via `todoExpiry.warningDays`
- Works in all languages with zero config
